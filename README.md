# vue2-webpack5

## 如何使用

-   `npm ci` 安装
-   `npm run serve` 启动 webpack 服务 chrome 输入 http://localhost:8888
-   `npm run build` 打包

## 本项目依赖

-   vue2
-   webpack5
-   tailwind
-   typescript
-   sass
-   less
